About the data model

In RD3, data is stored across many tables. An overview of the main tables of interest can be found below. If you are already familiar with the data model, you can view all tables. All reference tables and ontologies are stored in the ontologies database.

Table Description
Individuals An overview of the individuals (participants, patients, etc.) that are the focus of study. This includes demographics and clinical characteristics, as well as information of the data providers and references to related records within the database.
Pedigrees Overview of families (based on PED format). The definition of family members and the relationships between them are stored in Pedigree Members.
Clinical Observations Overview table containing information on clinical findings about an individual. Further details about clincial findings can be found in the Disease History and Phenotype Observations tables.
Samples Metadata on samples and their availability. There are also extensions for specific sample types.
Experiments Information about the experiments that were performed.
Files An aggregated view of files generated and where they are located. Specific locations and access information can be found in the File Locations and File Storage Locations tables.
Variant Interpretation Analyses An overview of the variant interpretations performed. Further information is stored in the Variant interpretations and Variant Interpretation Consensus. Variants are defined in the Genomic Variants table.
