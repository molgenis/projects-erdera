Data submission

Overview of the submission process

Before submitting data to RD3, register clinical and phenotypic metadata in PhenoStore. On a daily basis, RD3 will retrieve the latest data from PhenoStore including new submissions and updates to existing records.

There are two steps for submitting sample and experiment metadata to RD3. Please see our data submission manual for further instructions on the data submission and bulk import process.

  1. Enter data into one of the RD3 templates

    Several templates are available for submitting data to RD3. Each template is designed for a specific samples and experiment type. Download the relevant template and enter the data that you would like to submit to RD3. Templates will have required fields and optional fields, as well as dropdown lists that contain RD3-specific values. You may import as many samples and/or experiments as you like.

  2. Import data into RD3

    After completing the template, go to the Upload page, click the "browse" button to select your file, and click the "import" button.

    Once the import process has begun, a progress page will be displayed which will show the current status. If there are any errors, the import will stop and the error message will be displayed. Correct the error(s) and repeat the process until all errors are resolved.

    Note: Data will not be imported into RD3 if there are errors.
