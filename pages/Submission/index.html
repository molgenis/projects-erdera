Overview of the submission process

Before submitting data to RD3, register clinical and phenotypic metadata in PhenoStore. On a daily basis, RD3 will retrieve the latest data from PhenoStore including new submissions and updates to existing records.

There are a few ways for submitting sample and experiment metadata to RD3. Please see our data submission manual for further instructions.

  1. Enter data into one of the RD3 templates

    Download the relevant template(s) and enter information about the sample or experiments. There are required and optional fields.

  2. Import data into RD3

    Go to the Upload page and select the file you wish to import. Once the import process has begun, you will be shown the processing steps. If there are any errors, the import will stop and the error message will be displayed. Correct the error(s) and repeat the process until all errors are resolved.

    Note: Data will not be imported into RD3 if there are errors.
